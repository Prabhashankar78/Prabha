# Smart Mobile Store - React E-Commerce

A modern e-commerce website for premium smartphones built with React.

## Features

✅ Product browsing with 9 premium smartphones
✅ Shopping cart functionality
✅ Quantity selection for each product
✅ Dynamic cart updates
✅ Order form with customer details
✅ Smooth scrolling navigation
✅ Responsive design
✅ Modern UI with animations

## Installation

1. Install dependencies:
```
npm install
```

2. Start the development server:
```
npm start
```

3. Open http://localhost:3000 in your browser

## Build for Production

```
npm run build
```
